ENTERPRISE RESILIENCE (EDGE CASES)

Open-Ended Date Parsing ("Present" ranges)
Candidates often list '2023 - Present' for current degrees. A naive parser mistakenly extracts '2023' as graduation year. We implemented custom tokenization to set end_year to null, preventing false analytics flagging.
Anti-Data Bleed (Cross-Candidate Merges)
Recruiter CSVs often misalign rows. We implemented a Word-Intersection Matcher. If a CSV attempts to merge "Naveen V" data into "Kaviyadharshini M", the system blocks the merge entirely, quarantining the data.
Zero-Width Artifact Sanitation
Invisible OCR characters (\u200b) from PDF extraction silently fail standard email validation. The Ingestion layer sanitizes these artifacts at byte-level prior to processing.
Descoped: Scanned Images & Rate Limiting
Image-only PDFs are flagged gracefully (Tesseract OCR descoped). Live LinkedIn scraping is stubbed to prioritize core deterministic conflict resolution under time constraints.
